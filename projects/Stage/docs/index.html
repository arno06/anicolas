The basics

Librairy for canvas manipulation. It support an AS3 API like (Please Adobe do not sue me) 

var stage = new Stage(800, 600);
var shape = new Shape();
shape.beginFill("rgb(255, 0, 0)");
shape.drawRectangle(-50, -50, 100, 100);
shape.endFill();
stage.addChild(shape);

Features :

  • EventDispatcher
  • Event
  • MouseEvent
  • DisplayObject
  • Container
  • RequestAnim shim layer (thanks to Paul Irish)
  • Drawing API (beginFill, drawRectangle, drawCircle, moveTo, lineTo...)
  • FPS Meter
  • Extensible
